Workflow Guide

How to Analyze Model Papers and Answer "What Counts as Good Writing?"

A Step-by-Step Process for Research Students

Overview

This simplified workflow guides you through analyzing model papers to identify effective writing strategies. You'll reflect on what counts as good writing, analyze 2-3 model papers focusing on Task Achievement, and create an HTML report with actionable insights for your own writing.

Table of Contents

  • Step 1: Reflect & Prepare
  • Step 2: Select Papers & Analyze
  • Step 3: Create HTML Report
  • Step 4: Submit Your Work

1Reflect & Prepare

Files to work with: PhDagentSpring2026/writing/assessment/What_counts_as_good_writing_STUDENT.md
PhDagentSpring2026/writing/assessment/writing_instructions_formatted.md

What to do:

  • Reflect on good writing in your field:
    • Open What_counts_as_good_writing_STUDENT.md and answer the key questions
    • Identify 2-3 model papers from your target journals/conferences
    • Consider what makes writing "excellent" in your discipline
  • Review the assessment rubrics in writing_instructions_formatted.md:
    • Focus on Task Achievement criteria (Excellent/Satisfactory/Unsatisfactory)
    • Understand what "extremely clear" vs "generally clear" means
    • Note requirements for citations and referencing
  • Optional: Review Session 1 & 2 materials for rhetorical moves and synthesis strategies

Checklist:

I have identified my target journals/conferences
I have selected 2-3 model papers to analyze
I understand the Task Achievement criteria

2Select Papers & Analyze

Where to save papers: PhDagentSpring2026/literature/[YourFolderName]/
Reference example: PhDagentSpring2026/literature/GFY*/task_achievement_analysis.html

What to do:

  • Select 2-3 model papers from your target journals/conferences that demonstrate excellent Task Achievement
  • Analyze the papers focusing on key aspects:
    • Research Background: How do papers begin? What makes the background "extremely clear"?
    • Gap Identification: How do papers identify and articulate research gaps clearly?
    • Literature Synthesis: How do papers synthesize and critically evaluate literature?
    • Citations: How are sources integrated effectively?
  • Extract excerpts that demonstrate excellent Task Achievement
  • Compare patterns across papers to identify common strategies
  • Document your findings with specific excerpts and brief explanations

Example Analysis:

### Paper 1: [Title] **Research Background Excerpt:** "Graph Neural Networks (GNNs) have demonstrated remarkable effectiveness..." **Analysis:** - Uses evaluative language: "remarkable effectiveness" - Present perfect tense: "have demonstrated" - Multiple citations showing field knowledge - Clear progression from general to specific
Tip: You can start by analyzing just 2 papers focusing on Research Background and Gap Identification, then expand to more aspects if needed.

3Create HTML Report

Example output: PhDagentSpring2026/literature/GFY*/task_achievement_analysis.html

What to do:

  • Create an HTML file named task_achievement_analysis.html
  • Include in your report:
    • Header with paper information (titles, authors, venues)
    • For each aspect: excerpts and analysis from your papers
    • Key insights section: What patterns did you discover?
    • Actionable strategies: How will you apply these insights?
  • Make it visually clear: Use CSS styling, color-coding, and clear sections
  • Save the file in your literature folder: PhDagentSpring2026/literature/[YourFolderName]/task_achievement_analysis.html
Tip: View the example HTML file at PhDagentSpring2026/literature/GFY*/task_achievement_analysis.html for a complete working example.

4Submit Your Work

What to do:

  • Save your HTML file in the correct location: PhDagentSpring2026/literature/[YourFolderName]/task_achievement_analysis.html
  • In your Moodle reply, include:
    • The GitHub file path to your HTML file
    • A brief reflection on:
      • To what extent the analyses can inform your writing
      • Key insights you discovered
      • How you plan to apply these insights

Example Moodle Post:

File path: PhDagentSpring2026/literature/[MyName]/task_achievement_analysis.html Reflection: I analyzed two model papers focusing on Task Achievement (Research Background and Gap Identification). The analysis revealed that both papers use evaluative language strategically in centrality statements, and they employ different but equally effective strategies for gap identification‚Äîone uses systematic enumeration while the other uses concrete examples. This has helped me understand what "extremely clear" research background and "clearly delineated" gaps look like in practice. I plan to apply these insights by incorporating similar linguistic patterns and gap-signaling strategies in my own introduction and literature review.

Summary: The Complete Workflow

  1. Reflect & Prepare: Answer reflection questions and review assessment rubrics
  2. Select Papers & Analyze: Choose 2-3 model papers and analyze them focusing on Task Achievement
  3. Create HTML Report: Generate a report with excerpts, analysis, and actionable insights
  4. Submit: Post the file path and reflection in Moodle
