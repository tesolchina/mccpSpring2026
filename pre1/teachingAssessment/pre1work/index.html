Content: Key Findings

RQ1: Proxy vs. Single LLM

  • Proxy Metrics (SOG): Only capture ~5% correlation. They are too rigid.
  • Single LLM (Qwen2.5-7B): Reaches ~10.5% (Double the accuracy!).
  • Insight: Even a small LLM acts like a better "user" than a formula.

RQ2: Context Matters (Auxiliary Data)

  • Taobao Dataset: Adding "Purchase" data hurt performance (Dropped to 5.1%).
  • Serendipity-2018: Adding "Ratings" improved performance (Up to 12.3%).
  • Takeaway: More data isn't always better—it must be relevant.

Finding 3: What if we use multiple LLMs together?
6/10
Content: Key Findings

RQ3: Two Heads Are Better Than One

  • Method: Ensemble of 3 models (Qwen-14B, Qwen-72B, GPT-4).
  • Result: Accuracy jumps to over 20% correlation.
  • Why? Different models cancel out each other's biases.
