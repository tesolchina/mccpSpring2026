Tips for Success

  • Use a paper in machine-readable format (PDF, LaTeX, Word) - not scanned images
  • Focus on showing relationships between sections, not just listing them
  • Include brief text excerpts to show what each section contains
  • Consider using color coding for different types of content (methods, results, theory, etc.)
  • Moodle forum supports basic HTML - keep your code simple
